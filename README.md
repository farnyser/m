Order Matching Engine
=====================

**M** is a basic order matching engine (finance). 
It's a toy project to learn and play with order book.

It handles combination of:

- *Market* vs *Limit*
- *All or nothing* (AON) vs *Any part of*
- *Immediate or cancel* (IOC) vs *Day*

The project is using CMake and offer two target: 
*m* (random market performance test) and *test* (basic unit testing).
